## Demo

[➡️ 데모사이트 바로가기](https://instagram-clone-sangwon.com)

웹 사이트 방문 후 회원가입을 새로 하셔도 되지만 </br>
이메일 `Jimin_Ahn@google.com` </br>
비밀번호 `Qwer1234@` </br>
입력하신 후 로그인을 하면 더 편리하게 프로젝트를 감상하실 수 있습니다. </br></br>
현재 기존에 존재하는 서비스를 똑같이 클론해서 배포했다는 이유로 여러 브라우저에서 사기성 사이트로 인식하고 있습니다.
문제를 해결하기 위해 지속적으로 경고 해제 문의를 하고 있습니다. 
안전한 사이트이니 걱정 없이 방문하셔도 됩니다!
세부정보를 클릭 후 "이 안전하지 않은 사이트를 방문"을 클릭하시면 이용하실 수 있습니다!
빠른 시일 내에 해결하겠습니다!

<br/>

## Quick Start

```javascript
cd frontend
yarn install
yarn start
```
